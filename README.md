# utilsFun 实用工具类函数

### 使用

npm i @linlt/utilsfun

### 项目更新

1、npm login // 登录账号 <br />
2、npm whoami // 查看登录状态 <br />
3、npm publish --access public // 发布 <br />

### 目录结构

```
├─package.json	    //配置相关
├─index.js		      //入口文件
├─utils				      // 项目文件
  ├─array		              //数组类工具函数
  ├─common	              //公共函数
  ├─function              //方法类工具函数
  └─object		            //对象类工具函数

```
